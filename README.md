# Movie Tracker Database

A simple CLI (Command Line Interface) application built with Python and SQLite3 to manage a personal movie collection.
This project demonstrates basic CRUD (Create, Read, Update, Delete) operations and SQL database integration.

## Features

- Add new movies with title, release year, and rating.
- View the entire movie collection in a formatted list.
- Search for movies by title (using partial matches).
- Update the rating of an existing movie by its ID.
- Delete movies from the database.
- Error handling for invalid user inputs.

## Technologies Used

- Python 3
- SQLite3 (Standard library)

## How to Run

1. Make sure you have Python installed on your system.
2. Download or clone the repository.
3. Run the script using the following command:
   python movie_tracker.py

## Usage

Follow the on-screen menu to interact with the database:
1. Select '1' to add a movie.
2. Select '2' to view all movies.
3. Select '3' to search for a movie.
4. Select '4' to delete a movie.
5. Select '5' to update a rating.
6. Select '6' to exit the application.
